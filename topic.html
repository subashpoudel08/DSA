üöÄ Problem: Max Sum of Window Size K

Given an array and an integer k, find the maximum sum of any subarray of size k.

üî¢ C++ Code

int maxSum(vector<int> arr, int k) { int max_sum = 0, window_sum = 0; for (int i = 0; i < k; i++) window_sum += arr[i]; max_sum = window_sum; for (int i = k; i < arr.size(); i++) { window_sum += arr[i] - arr[i - k]; max_sum = max(max_sum, window_sum); } return max_sum; }
