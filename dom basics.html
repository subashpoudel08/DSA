JavaScript DOM Targeting Methods | Target by Id, Class, Tag

How to reliably locate specific elements in the Document Object Model to inspect or manipulate them.

DOM Selectors HTMLCollection document Beginner Console
Reading time: ~10 min Keywords: DOM, getElementById, getElementsByClassName, getElementsByTagName, document, HTMLCollection

Introduction

The HTML DOM represents a page as a tree of nodes (elements, attributes, and text). JavaScript can target any node to read or change it. When pages contain multiple similar elements, you need precise methods to select the right one.

Summary

  • Three primary methods: document.getElementById(id), document.getElementsByClassName(name), document.getElementsByTagName(name).
  • By Id returns a single element (or null) because an id should be unique.
  • By Class/Tag returns an HTMLCollection (arrayâ€‘like) of matching elements; access with indices.
  • document exposes useful entry points like head, title, body, links, images, and more.
  • Pass ids, class names, or tag names as strings; do not prefix ids with # or classes with . in these methods.

Choosing a Method

Select the targeting approach that matches your intent:

  • Use Id for a single, unique element. Returns the element or null.
  • Use Class when multiple elements share the same role or style. Returns an HTMLCollection.
  • Use Tag to fetch all elements of a type (e.g., all li or ul). Returns an HTMLCollection.
  • Use document.* properties for quick access to global parts like head, body, links, or images.
Arguments are plain strings. Do not prefix with # for ids or . for classes in these methods.

Syntax & Patterns

Target by Id

const el = document.getElementById("header");
// el is the single element with id="header" or null if not found

Target by Class Name

const items = document.getElementsByClassName("list");
// items is an HTMLCollection (array-like)
const firstItem = items[0];

Target by Tag Name

const uls = document.getElementsByTagName("ul");
// Get the third <ul> if it exists
const thirdUl = uls[2];

Work with HTMLCollection

// Option 1: traditional for loop
const links = document.links;
for (let i = 0; i < links.length; i++) {
  console.log(links[i].href);
}

// Option 2: convert to Array to use array helpers
Array.from(document.images).forEach(img => console.log(img.src));

If you expect a single element and your selection returns a collection, pick an index like [0] or filter after converting to an array.
