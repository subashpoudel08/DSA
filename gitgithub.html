🚀 Git & GitHub Study Guide

Bhai Subash ke liye ekdum beginner-friendly guide!

📌 What is Git?

Git ek version control system hai jo developers ko apne code ke versions track karne deta hai. Jaise time machine for your code!

📌 What is GitHub?

GitHub ek online platform hai jaha tu apna git repository upload kar sakta hai. Team ke sath collaboration bhi aasaan hota hai.

⚙️ Common Git Commands

  • git init: New repo create karta hai
  • git add .: Saare files stage karta hai
  • git commit -m "message": Changes ko save karta hai
  • git remote add origin <url>: GitHub repo se connect
  • git push -u origin master: Files push karta hai GitHub pe

📁 .gitignore

Yeh file specify karti hai ki kaunse files Git me track nahi hone chahiye.

.env
node_modules/
build/
*.log

🔐 .env File

Isme sensitive info jaise API keys hoti hain. Yeh kabhi bhi GitHub pe push nahi karna chahiye.

🌿 main vs master

master pehle default branch hoti thi, ab main default hai modern repos me.

🌍 CMD Se GitHub Pe Push Ka Flow

  1. git init
  2. git add .
  3. git commit -m "Initial commit"
  4. git remote add origin https://github.com/user/repo.git
  5. git push -u origin main (ya master)

🚫 Already pushed file ko hide kaise kare (without delete)

  1. File ko .gitignore me add karo
  2. Phir Git se untrack karo: 
    git rm --cached file_name
  3. Phir commit aur push: 
    git commit -m "removed from tracking"
git push
