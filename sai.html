📌 Insertion Sort

Ek-ek element uthao aur use sorted list me sahi jagah insert karo. Jaise cards sort karte hai haath me 🃏

void insertionSort(vector<int>& arr) { for (int i = 1; i < arr.size(); i++) { int key = arr[i]; int j = i - 1; while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j--; } arr[j + 1] = key; } }

📌 Selection Sort

Minimum element dhundo aur current position pe daal do. Ek-ek pass me ek element apni sahi jagah pe chala jata hai ✅

void selectionSort(vector<int>& arr) { for (int i = 0; i < arr.size() - 1; i++) { int minIndex = i; for (int j = i + 1; j < arr.size(); j++) { if (arr[j] < arr[minIndex]) minIndex = j; } swap(arr[i], arr[minIndex]); } }

📊 Complexity Comparison

Algorithm Best Case Average Case Worst Case Space Stable?
Insertion Sort O(n) O(n²) O(n²) O(1) ✅
Selection Sort O(n²) O(n²) O(n²) O(1) ❌

🧠 Use Cases

  • ✔ Insertion Sort: Jab array already sorted ho ya chhoti size ka ho (like 20-30 elements)
  • ✔ Selection Sort: Jab swaps kam matter karein, aur stable hona zaroori na ho

📚 Summary

  • Insertion Sort: Best for almost sorted arrays
  • Selection Sort: Simple to understand but inefficient for large arrays
  • Dono in-place sorting algorithms hai (O(1) space)
  • Insertion Sort = Stable, Selection Sort = Not Stable
