Largest Rectangle in Histogram | Stacks and Queues

Find the largest rectangle area in a histogram using stack techniques.

Stack Monotonic Stack Arrays O(N)

Introduction

The problem is to find the largest rectangle in a given histogram.

  • You are given an array of integers, each representing the height of a bar.
  • The width of every bar is constant and assumed to be 1.

Example & Breakdown

Example histogram heights: [2, 1, 5, 6, 2, 3] at indices 0 to 5.

  • The task is to return the area of the largest rectangle that can be formed.
  • A rectangle is a continuous set of bars, with height equal to the minimum among them.
  • Example 1: height 1 at index 1 spans 6 bars â‡’ area = 6 Ã— 1 = 6.
  • Example 2: height 2 can span indices 2..5 â‡’ width = 4 â‡’ area = 4 Ã— 2 = 8.
  • Example 3 (largest here): bars of heights 5 and 6 â‡’ width = 2, height = 5 â‡’ area = 10.

Brute-Force Solution

  • Iterate each bar as the limiting height; expand left and right while bars â‰¥ current height.
  • Width = number of included bars; area = height Ã— width; track maximum.

PSE & NSE (Two-Pass)

For each bar heights[i], find boundaries:

  • PSE: first index to the left with smaller height; use -1 if none.
  • NSE: first index to the right with smaller height; use n if none.
  • Width = NSE - PSE - 1, area = heights[i] Ã— width.
  • Three passes: compute PSE, compute NSE, then compute max area â‡’ O(N) time, O(N) space.

Optimal Single-Pass

  • Maintain a stack of indices with increasing heights.
  • When current height is smaller than stack topâ€™s height, pop and compute area for that bar.
  • For a popped index p: height = heights[p], right = current index, left = new stack top (or -1), width = right - left - 1.
  • After the scan, pop remaining bars using right = n.

Final Algorithm & Pseudocode

def largestRectangleArea(heights):
    n = len(heights)
    stack = []  # store indices of increasing heights
    max_area = 0
    for i in range(n + 1):  # include sentinel at n
        curr = heights[i] if i < n else 0
        while stack and (i == n or curr < heights[stack[-1]]):
            h = heights[stack.pop()]
            left = stack[-1] if stack else -1
            width = i - left - 1
            max_area = max(max_area, h * width)
        stack.append(i)
    return max_area  # O(N) time, O(N) space

Each index is pushed and popped at most once â‡’ O(N) time; stack uses O(N) space worst-case.
