Maximal Rectangle | Stacks and Queues

Find the area of the largest rectangle of 1s in a binary matrix by reducing rows to histograms.

Matrix Histogram Stack O(NÃ—M)

Introduction

Maximal Rectangle problem on a binary matrix.

  • Find the area of the largest rectangle that contains only 1s.
  • Prerequisite: understand how to compute the Largest Rectangle in Histogram; we will reuse that solution.

Example & Breakdown

Problem: Given an N Ã— M matrix with entries in {0,1}, return the area of the largest rectangle consisting only of 1s.

  • Example 1: a highlighted rectangle of size 1Ã—5 with all 1s â‡’ area = 5.
  • Example 2: a rectangle of size 2Ã—3 with all 1s â‡’ area = 6.
  • Goal: maximize area over all valid rectangles; for the drawn example, the answer is 6.
  • Constraint: any 0 inside a candidate region invalidates it as a rectangle of 1s.

Brute-Force Solution

  • Reduce the 2D problem to a sequence of 1D histogram problems, one per row.
  • For row i, the height of column j equals the number of consecutive 1s above (including row i).
  • Height rule at (i, j):
    • If matrix[i][j] == 0, height = 0.
    • If matrix[i][j] == 1, height = 1 + height(i-1, j).
  • Row 0: histogram is just the first row (1 â†’ 1, 0 â†’ 0). For example [1,0,1,0,1], max area = 1.
  • For each rowâ€™s histogram, compute the largest rectangle area and track the global maximum.

PSE & NSE (Two-Pass)

Connecting the matrix problem to the histogram solution:

  • Iterate each row of the matrix; construct its histogram heights array.
  • Apply the Largest Rectangle in Histogram algorithm to that array.
  • Maintain the maximum area seen across all rows; this is the final answer.

Optimal Single-Pass

  • How to build histograms efficiently for each row?
  • Naive approach: for every cell, scan upwards to count consecutive 1s (inefficient).
  • Optimized approach: reuse the previous rowâ€™s heights â€” increment on 1, reset to 0 on 0.
  • This behaves like a prefix-sum style accumulation per column.

Dry Run

Example walk-through of row-wise histogram construction and areas:

  • Row 0: matrix row [1, 0, 1, 0, 1] â‡’ heights [1, 0, 1, 0, 1] â‡’ max area = 1.
  • Row 1: matrix row [1, 1, 1, 1, 1] â‡’ heights [2, 1, 2, 1, 2] (built on previous row).
  • Row 2: matrix row [1, 1, 1, 1, 1] â‡’ heights [3, 2, 3, 2, 3] â‡’ largest rectangle area found here is 6.
  • Row 3: matrix row [1, 0, 0, 1, 0] â‡’ heights [4, 0, 0, 3, 0] (zeros reset the count).

Final Algorithm & Pseudocode

def maximalRectangle(matrix):
    n = number of rows in matrix
    m = number of columns in matrix
    if n == 0 or m == 0: return 0

    heights = array of length m initialized to 0
    maxArea = 0

    for i from 0 to n-1:
        for j from 0 to m-1:
            if matrix[i][j] == 1:
                heights[j] = heights[j] + 1
            else:
                heights[j] = 0

        # prerequisite function from Largest Rectangle in Histogram
        area = largestHistogramArea(heights)
        maxArea = max(maxArea, area)

    return maxArea  # overall O(NÃ—M)

Build heights row by row in O(M) per row; largestHistogramArea runs in O(M). Total time O(NÃ—M). Space O(M) extra; or O(NÃ—M) if using an explicit pSum matrix.

C++ Implementation

class Solution {
public:
    int largestRectangleArea(vector<int>& heights) { // prerequisite
        int n = (int)heights.size();
        vector<int> st; // monotonic increasing stack of indices
        int maxarea = 0;
        for (int i = 0; i <= n; i++) {
            int curr = (i == n ? 0 : heights[i]);
            while (!st.empty() && heights[st.back()] > curr) {
                int hIdx = st.back(); st.pop_back();
                int left = st.empty() ? -1 : st.back();
                int right = i;
                int width = right - left - 1;
                maxarea = max(maxarea, heights[hIdx] * width);
            }
            st.push_back(i);
        }
        return maxarea;
    }

    int maximalRectangle(vector<vector<int>>& matrix) {
        if (matrix.empty() || matrix[0].empty()) return 0;
        int n = (int)matrix.size();
        int m = (int)matrix[0].size();
        vector<int> heights(m, 0);
        int best = 0;
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < m; ++j) {
                if (matrix[i][j] == 1) heights[j] += 1; else heights[j] = 0;
            }
            best = max(best, largestRectangleArea(heights));
        }
        return best;
    }
};
